Sales Data Analysis Project

Overview

This project analyzes sales data using Python to extract useful insights and generate visual reports.

Features

- Data cleaning using pandas
- Data analysis (revenue by product and customer)
- Data visualization using matplotlib

Project Structure

- data/raw → original data
- data/processed → cleaned data
- src → python scripts
- outputs → charts

Tools Used

- Python
- Pandas
- Matplotlib

Results

The project generates:

- Cleaned dataset
- Revenue analysis
- Visual charts

Business Insights

From the analysis:

- Laptop generates the highest total revenue, making it the most valuable product.
- Sara is the most valuable customer based on total spending.
- Sales show growth in February compared to January, indicating a positive trend.

Recommendation

- Focus marketing efforts on Laptop products
- Retarget high-value customers like Sara
- Analyze the reason behind February growth and replicate it
